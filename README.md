## 🎈最佳实践

`*.md` 为笔记文件，新建文件，或向已有文件添加内容后保存，执行以下命令，提交至本地git仓库

```shell
git add .

# commitment description是对于本次提交的描述，例如“新增了生态建设方面的内容”
git commit -m 'commitment description' 
```

提交后，执行以下命令，推送至远端仓库以便共享

```shell
git push
```
